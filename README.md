### 翻译和维护
请先安装po4a。

只要翻译`po`目录下面的文件就可以了。推荐使用专门的软件，比如`lokalize`。翻译好之后，请务必预览你翻译好的命令。
使用`./preview command`就可以预览。

**注意**：
1. 对于不了解的内容，请**不要**翻译。

翻译规范参考[wiki](https://github.com/man-pages-zh/wiki/wiki/%E7%BF%BB%E8%AF%91%E8%A7%84%E8%8C%83)。
